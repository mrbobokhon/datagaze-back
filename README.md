# django-datagaze-project



## Getting started

1. ```python3 -m venv venv```

2. ``` .venv/bin/activate```

3. ```pip install -r requirements/develop.txt```

4. ```cp .env.example .env```

5. ```python3 mananage.py migrate```

6. ```python3 manage.py runserver```
